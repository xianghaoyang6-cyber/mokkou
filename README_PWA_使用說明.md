# 我の木工 PWA v23 穩定版

這版以可正常使用的 v17 為底重做，不含啟動畫面。

新增：專案管理、購物清單、耗材/五金庫、板材排版鋸路設定、錯誤提醒、一鍵備份/匯入。

部署時請上傳「woodworking_pwa_v23」整個資料夾，且資料夾內需直接包含 index.html、manifest.json、service-worker.js、version.json、icons/。

如果手機仍讀到舊版，先開 clear-cache.html 清除 PWA 快取。


## v25 穩定保護版

- 以 v23 穩定版為底。
- 新增首頁版本顯示。
- 新增 7 天備份提醒與首頁「立即完整備份」按鈕。
- 新增「檢查更新」、「清除快取」、「重新載入最新版」按鈕。
- service-worker 改為導覽頁 network-first，降低舊版快取卡住機率。
- 無啟動畫面。


## v25 專案強化版
- 專案複製
- 專案搜尋
- 狀態篩選
- 成本統計
- 專案備註欄位加大
- 保留 v24 穩定保護與備份功能。


## v26 更新
- 新增「步驟」分頁：依目前家具類型產生施工步驟，可勾選、備註、上移、下移、刪除與匯出文字。
- 新增「紀錄」分頁：可保存材料、下料、組裝、完成、問題與改善紀錄，支援本機壓縮照片。
- 版本更新為 PWA v26，無啟動畫面。
