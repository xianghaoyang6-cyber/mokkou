# 手機驗收清單 v11.13

## 一、版本顯示

1. 首頁上方顯示「我の木工 PWA v11.13」。
2. 標籤顯示「PWA v11.13 FOLD STABLE」。
3. 頁尾顯示 v11.13。
4. 不應再看到首頁顯示 v11.11。

## 二、大型展開 / 收合按鈕

請逐頁檢查大型區塊右側按鈕：

1. 首頁
2. 計算
3. 施工圖
4. 排版圖
5. 參數庫
6. 備份
7. 旋鈕頁面，如目前版本有顯示

每頁檢查：

1. 收合狀態顯示「展開」。
2. 點開後顯示「收合」。
3. 再點一次後回到「展開」。
4. 切到別頁再切回來，按鈕文字不消失。
5. 不應出現空白膠囊。
6. 不應只在點擊當下才出現文字。

## 三、小型功能按鈕

確認以下小按鈕不受影響：

1. 計算
2. 看結果
3. 下載 SVG
4. 下載 DXF
5. 下載 PNG
6. 備份 / 還原
7. 回頂部

## 四、資料安全檢查

1. 原本輸入的尺寸資料仍存在。
2. localStorage 主要資料未被清除。
3. 照片庫仍可開啟。
4. IndexedDB 照片庫不應被清快取頁刪除。
5. 備份 ZIP / 還原功能不應被影響。

## 五、通過標準

v11.13 驗收通過條件：

1. 版本顯示正確。
2. 大型展開 / 收合按鈕在切頁後仍穩定顯示。
3. 沒有空白膠囊。
4. 小型功能按鈕正常。
5. 計算、施工圖、SVG / DXF / PNG 輸出流程正常。
